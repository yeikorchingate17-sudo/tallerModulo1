Descripción del Proyecto:
Este taller contiene un análisis de dataset centrado en indicadores de salud mundial. El objetivo es realizar practicas de tecnologias como paython y pandas, identificando patrones, funciones, filtros con una tematica de el desarrollo económico y la salud mundial, el gasto en salud y la prevalencia de enfermedades crónicas como la obesidad y la diabetes.

Información del Dataset
Fuente: Datos globales de indicadores de salud (archivo salud_mundial.csv).

Contenido: El dataset incluye información de 159 países.

Variables clave: * pais y region.

esperanza_vida (variable objetivo para categorización).

gasto_salud_usd (indicador económico).

obesidad_pct y diabetes_pct (indicadores de salud pública).
