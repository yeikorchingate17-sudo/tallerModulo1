Descripcion del proyecto:

En este proyecto se realizan varios ejercicios donde evaluamos los conceptos aprendidos en el modulo uno.

En el ejercicio 1.1 se realizan pruebas de conocimientos basicos en donde se evalua apectos importates como los bucles for, los condicionales if, la integracion de variables y la logica de programacion. Aqui realizamos calculos de promedio, maximo y minimo.
